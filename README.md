# Expense-Tracker

Expense Tracer is a console-based Python application designed to help users
manage and track daily expenses efficiently.  
The project uses a menu-driven approach and focuses on clean logic and
beginner-friendly implementation.

## Key Features
- Add expenses with date, category, description, and amount
- View all recorded expenses in an organized format
- Calculate total spending
- Simple and user-friendly menu system
- Lightweight and terminal-based application

## Technologies Used
- Python
